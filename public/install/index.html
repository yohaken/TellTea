ดาวน์โหลดไฟล์ติดตั้ง
  1. เช็คเลขเวอร์ชันด้านบนก่อนกดดาวน์โหลดทุกครั้ง
  2. เปิดด้วย Chrome บนแท็บเล็ต/เอมูเลเตอร์ → กดดาวน์โหลด → อนุญาตติดตั้ง
  3. ถ้าขึ้นติดตั้งไม่ได้: ถอน nPos-telltea เดิม แล้วกลับมากดดาวน์โหลดอีกครั้ง (ทำครั้งเดียว)
  4. เปิดไอคอน nPos-telltea แล้วเทียบเลขเวอร์ชันในแอป
  5. แมนิเฟสต์: /downloads/latest.json
