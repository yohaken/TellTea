ดาวน์โหลดไฟล์ติดตั้ง

สแกนด้วยกล้องมือถือ/แท็บเล็ต แล้วเปิดลิงก์ใน Chrome

https://telltea-pos.web.app/install/

  1. เช็คเลขเวอร์ชันด้านบนก่อนกดดาวน์โหลดทุกครั้ง
  2. เปิดด้วย Chrome บนแท็บเล็ต → กดดาวน์โหลด → อนุญาตติดตั้งจากแหล่งนี้
  3. ถ้าติดตั้งทับไม่ได้: ถอน nPos-telltea เดิม แล้วกลับมากดดาวน์โหลดอีกครั้ง
  4. เปิดไอคอน nPos-telltea → เข้างาน → จะเห็นเมนูหน้าร้าน (ขาย / รอบ / ประวัติ / ตั้งค่า)
  5. ในแอปมีปุ่มอัปเดตเอง — ไม่ต้องสแกน QR ทุกครั้งเมื่อมีเวอร์ชันใหม่
