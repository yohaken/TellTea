## รายละเอียดเฟสย่อยที่วางไว้เยอะ

### ทำแล้ว (อย่าทำซ้ำ)

| เฟส | ของที่ส่ง |
|------|-----------|
| N0–N5 | อัปเดต APK · จอ · heartbeat · ปริ้น USB/BT · ลิ้นชัก · ops/diagnose |
| N6.0–6.5 | เข้างาน · เมนู · ตะกร้า · เงินสด/PP · outbox · รอบ · ใบเสร็จ |
| N6.6–P2 | parity ขายหลักเกือบครบเว็บ |

### P3 — ก่อนส่งเครื่องจริง (รอบนี้)

| # | งาน | ทำไมสำคัญพรุ่งนี้ |
|---|-----|---------------------|
| P3.1 | LAN printer TCP `:9100` | ร้านที่ใช้ปริ้น Wi‑Fi ไม่มี USB |
| P3.2 | พักบิล local («ส่งค้างไว้») | พนักงานสลับออเดอร์ได้แม้เว็บยัง stub |
| P3.3 | ป้ายคิวซิงก์ + กดรีซิงก์ | เห็นบิลค้างเมื่อเน็ตหลุด |
| P3.4 | สรุปรอบสั้นบนจอขาย | ไม่ต้องพิมพ์ X ทุกครั้งเพื่อดูยอด |
| P3.5 | เอกสารนำร่องวันจริง | สคริปต์เทส 15 นาที |

### P4 — วันนำร่อง (พรุ่งนี้ · คน ไม่ใช่โค้ด)

1. ติดตั้งจาก https://telltea-pos.web.app/install/  
2. ตั้งค่าปริ้น (USB/BT **หรือ** LAN) + พิมพ์ทดสอบ + ลิ้นชัก  
3. เปิดรอบ → ขายเงินสด 1 · PromptPay 1 · ของหมด 1 · ส่วนลด 1  
4. พิมพ์ซ้ำ · ทำลายบิล · สรุปกลางรอบ · ปิดรอบ  
5. ปิด Wi‑Fi ชั่วคราว → ขาย 1 บิล → เปิดเน็ต → ดูซิงก์  
6. บันทึก feedback ในไทม์ไลน์ ops / โน้ตสั้น  

**ห้าม** ตัดเว็บขายในวันนี้

### P5 — หลัง feedback (1–2 รอบ APK)

- แก้พิมพ์ไม่ติด / QR / ทอน / ของหมด ตามรายงานจริง  
- ข้อความพนักงานให้สั้นลงถ้าสับสน  
- ปรับ Z/X ถ้าตัวเลขไม่ตรงความรู้สึกหน้าร้าน  

### P6 — ซิงก์/หลังร้านให้แน่น

- `nposVoidSale` CF (ทำลายบิลสะท้อน Firestore ถ้าต้องการ)  
- หน้าคิวซิงก์ละเอียด (retry ทีละบิล)  
- รายงานยอด nPos ในหลังร้านเทียบ local  

### P7 — admin บนเครื่อง (ทำเมื่อจำเป็นเท่านั้น)

- แก้ราคาเร็ว / ของหมดจากรายการเต็ม  
- **ไม่** พอร์ตครอปรูปทั้งก้อนถ้าเบราว์เซอร์ใช้ได้  

### P8 / N7 — ตัดเว็บขาย

เกณฑ์ผ่านก่อนตัด:

- [ ] นำร่อง ≥ 1 วันเต็มโดยไม่ต้องเปิด `/pos/sell`  
- [ ] พิมพ์ + ลิ้นชัก + PromptPay ผ่านบนเครื่องจริง  
- [ ] ops log อ่านได้โดยไม่ถามพนักงาน  
- [ ] เจ้าของโอเคว่าเมนูตั้งบน `/pos/menu/` อย่างเดียว

## ช่องทาง deploy พรุ่งนี้

1. Merge PR → GitHub Actions ビルド APK → `latest.json`  
2. แท็บเล็ต: ตั้งค่า → อัปเดต หรือเปิดหน้าติดตั้ง  
3. ยืนยันเวอร์ชันในแอป = **1.10.0 (14)** ก่อนเริ่มขายจริง
