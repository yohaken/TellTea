## โครงใหญ่ (เฟส)

```
A  เข้าสู่ระบบ + ลิงก์ทางเข้า
B  โครงหน้า / แท็บ
C  หมวด (เมนูอาหาร)
D  รายการเมนู (เพิ่ม · แก้ · รูป · ราคา · ตัวเลือก)
E  กลุ่มตัวเลือก + ผูกเมนู
F  ตั้งราคา (ตาราง)
G  โปรโมชั่น (stub)
H  ซิงก์ไปขาย (nPos) — ควันไฟหลังแก้
```
