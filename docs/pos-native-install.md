# TellTea POS — ลงแอปด้วยลิงก์ดาวน์โหลด

> วิธีง่ายสุดสำหรับร้าน: เปิดลิงก์บน Chrome แท็บเล็ต → ดาวน์โหลด `.apk` → ติดตั้ง

## ตัวตนหน้าร้าน (ไม่แชร์รหัสอีเมล)

| ชั้น | คืออะไร | ใครใช้ |
|------|---------|--------|
| อีเมลร้าน / Google เจ้าของ | ยืนยันร้าน หลังบ้าน | เจ้าของเท่านั้น — **ไม่ให้พนักงานรู้รหัส** |
| ตัวตนเครื่อง POS | แท็บเล็ตล็อกอินอัตโนมัติ (`posDevice`) | เครื่องเอง ไม่ใช่คน |
| พนักงานขาย | PIN / กะ | คนที่เปิดขาย |
| ลิงก์จากร้าน | เจ้าของใส่ที่ตั้งค่า → เครื่องอ่านที่เมนู «ลิงก์จากร้าน» | ส่งดาวน์โหลด/อัปเดตโดยไม่ต้องเข้าอีเมล |

หลักทั่วไปของ POS: พนักงานล็อกอินที่เครื่องด้วยตัวตนเครื่อง/PIN — **ไม่ใช่การเข้า inbox อีเมลร้าน**

## ลิงก์ที่ใช้ (คงที่)

| ใช้ทำอะไร | URL |
|-----------|-----|
| **หน้าดาวน์โหลด** (แนะนำ) | https://telltea-pos.web.app/install/ |
| ไฟล์ APK ตรงๆ | https://telltea-pos.web.app/downloads/telltea-pos.apk |
| เว็บ POS สำรอง (ยังไม่ลงแอป) | https://telltea-pos.web.app/pos/ |

ลิงก์หน้าดาวน์โหลดก็อยู่ที่ **ตั้งค่าหลังบ้าน → เครื่อง POS** (คัดลอกได้)

## ลงบนแท็บเล็ตร้าน (พนักงาน / เจ้าของ)

1. เปิด **Chrome** บนแท็บเล็ต
2. ไปที่ `https://telltea-pos.web.app/install/`
3. กด **ดาวน์โหลดไฟล์ติดตั้ง**
4. ถ้า Android ถาม — อนุญาตติดตั้งจาก Chrome / แหล่งนี้
5. เปิดไฟล์ `telltea-pos.apk` → ติดตั้ง → เปิดไอคอน **TellTea POS**

ไม่ต้องมีแชทส่วนตัว: พิมพ์ลิงก์เอง, บุ๊กมาร์ก, หรือทำ QR วางหน้าร้านก็ได้

---

## ไฟล์ APK มาจากไหน

หลัง merge เข้า `main` แล้ว GitHub Actions จะ:

1. บิลด์เว็บ + sync Capacitor
2. สร้าง debug APK
3. วางไฟล์ที่ `out-pos/downloads/telltea-pos.apk`
4. Deploy Hosting `telltea-pos`

ดังนั้นหลัง deploy สำเร็จ ลิงก์ด้านบนใช้งานได้ทันที

อัปเดตแอปเปลือกครั้งถัดไป: deploy ใหม่ → ไฟล์ที่ลิงก์เดิมถูกทับ (URL ไม่เปลี่ยน)

---

## ทางเลือก: ลงด้วย USB / Android Studio

ถ้าต้องการลงตรงจากคอม:

```bash
git pull origin main
npm ci
npm run build
npx cap sync android
npx cap open android   # Run ▶ บนแท็บเล็ตที่เสียบ USB
```

หรือ Build APK ใน Android Studio แล้ว `npm run publish:pos-apk` ก่อน deploy

## เช็คว่าลงถูก

- ไอคอนชื่อ **TellTea POS**
- เปิดแล้วเข้าหน้าขายได้
- เวอร์ชันมุมจอขึ้น **POS 58+**
- หลังบ้าน (ตั้งค่า → เครื่อง POS) เห็นโหมด **APK**

ถ้าเปิดแล้วขาว/ค้าง: แท็บเล็ตมีเน็ตไหม (รอบนี้แอปยังดึงเว็บจาก Hosting) → ปิดแอปแล้วเปิดใหม่

รายละเอียดเฟส native: `docs/pos-native-shell.md`
