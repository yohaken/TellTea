## แคตตาล็อก

- ไม่มีเมนู **ไมโล** — เพิ่มเมนู + ผูกกลุ่มหวาน/ท็อปตามมาตรฐานร้าน
- มาตรฐานชื่อ alias: **บุกบราวน์** / **ซอสบราวน์** / **ชีส** (รวมสะกด ทัอปปิ้ง → ท็อปปิ้ง ตอนบันทึก)
- กลุ่มที่ใช้ 0 เมนู → เก็บเข้าคลังหรือลบถาวรหลังยืนยัน
- หมวด orphan FoodStory → เก็บแล้ว / ลบถาวรจากตัวกรอง «เก็บแล้ว»
