## ระบบเช็คงาน

```bash
# ชุดตรวจโค้ด + smoke แคป (ต้องเน็ต / หลัง deploy)
node scripts/check-npos-shop.mjs

# ข้าม smoke แคปสด (CI โลคัลเร็ว)
SKIP_CAPTURE_SMOKE=1 node scripts/check-npos-shop.mjs
```

ผ่านเมื่อทุกขั้นในรายงานพิมพ์ `OK` และจบด้วย `OK check-npos-shop`

รายงานเขียนที่ `/opt/cursor/artifacts/npos-shop-check/report.json` (ถ้ามี path)

## มือร้าน (หลังอัปเดต APK)

| # | ผ่านเมื่อ |
|---|-----------|
| 1 | ตะกร้าโชว์ความหวาน/ท็อปปิ้งใต้ชื่อเมนู |
