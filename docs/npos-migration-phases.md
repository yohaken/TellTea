# nPos — เฟสย้ายจาก POS Web → Native

เป้าหมาย: ค่อยๆ ย้ายความสามารถจาก `telltea-pos` (WebView/Capacitor เดิม) มาแอป **nPos-telltea** โดยมีระบบเทสฟังก์ชันในเครื่อง + รายงานกลับหลังร้าน

## หลักการ

- **เว็บ POS เดิมยังไม่ลบ** จนกว่า native ผ่านเทสทีละฟังก์ชัน
- แต่ละเฟสต้องมี: สิ่งที่ตรวจในแอป · สิ่งที่ส่งกลับหลังร้าน · เกณฑ์ผ่าน
- อัปเดต APK ผ่าน `latest.json` (เวอร์ชันชัด + ลิงก์ติดตั้ง)

## เฟส

| เฟส | ชื่อ | สิ่งที่ทำใน native | เทส/รายงาน | เกณฑ์ผ่าน |
|-----|------|-------------------|------------|----------|
| **N0** | เปลือก + อัปเดต | Hello World, เวอร์ชัน, อัปเดต APK, ลิงก์ติดตั้ง | ติดตั้ง/อัปเดตได้ | ✅ ทำแล้ว (≤1.2.1) |
| **N1** | ตรวจเครื่อง | จอ 1,2,… · USB/BT/เน็ต | ส่ง snapshot กลับหลังร้าน (หมวดพับ) | ✅ ทำแล้ว (1.3.0) |
| **N2** | ตัวตนเครื่อง | installId → `posDevices` + heartbeat | หลังร้านเห็นเครื่อง online จาก native | ✅ รอบนี้ (1.4.0) |
| **N3** | จอหน้าร้านจริง | Presentation ยอด/ตะกร้าทดสอบ | ปุ่มเทส “ส่งไปจอ 2” | เห็นเลข/ยอดบนจอลูกค้า |
| **N4** | ปริ้นเตอร์เทส | ค้นหา + พิมพ์หน้าทดสอบ | สถานะ printerReady ส่งกลับ | พิมพ์ข้อความทดสอบได้ |
| **N5** | ลิ้นชัก | kick ผ่านปริ้นเตอร์ | ปุ่มเทสเปิดลิ้นชัก | ลิ้นชักเปิดได้ |
| **N6** | เมนู + ขาย offline | อ่านเมนู / บิลทดสอบ local-first | รายงานบิลเทส | ขายเทสครบรอบโดยไม่พึ่ง WebView |
| **N7** | ปิด WebView POS | ตัดเส้นทางเว็บขายหน้าร้าน | — | ร้านใช้ nPos เป็นหลัก |

## ระบบเทสฟังก์ชัน (ใช้ซ้ำทุกเฟส)

ในแอปมีหน้า **ตรวจเครื่อง / เทสฟังก์ชัน**:

1. รันเทสในเครื่อง (จอ / ฮาร์ดแวร์ / ปริ้น ฯลฯ)
2. กด **ส่งผลกลับหลังร้าน**
3. หลังร้านเปิดหมวดพับดูผลล่าสุดต่อเครื่อง

โครงรายงาน (collection `nposDiagnose/{installId}`):

```json
{
  "installId": "...",
  "reportedAt": 0,
  "versionCode": 5,
  "versionName": "1.3.0",
  "summary": "จอ 2 · เชื่อมต่อ 5",
  "displays": [{ "number": 1, "displayId": 0, "primary": true, "name": "..." }],
  "hardware": [{ "category": "USB", "title": "...", "detail": "..." }],
  "source": "npos-telltea"
}
```

## หลังร้าน

- หน้า **POS → จัดการ Pos**
  - หมวดพับ **เครื่อง nPos (ออนไลน์)** — รหัสเครื่อง + สถานะออน (N2)
  - หมวดพับ **ตรวจเครื่อง (nPos)** — ผลสแกนจอ/ฮาร์ดแวร์ (N1)

## นอกขอบเขตเฟสนี้

- ยังไม่สั่งพิมพ์/เปิดลิ้นชักจริง (N4–N5)
- ยังไม่ทำจอลูกค้ายอดขายจริง (N3)
